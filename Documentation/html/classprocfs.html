Constructor & Destructor Documentation

procfs::procfs ( )

This constructor creates an object of the procfs class.

The procfs object reads the /proc/cpuinfo file and gets all the required CPU info by calling the readNumOfCPUs and updateCPUDetails metthods. It also initializes a timer and connect the timeout singal to private slot member of this class: onTimerTimeout()
